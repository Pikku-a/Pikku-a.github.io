How to upload local repository to github


Credit to the person who wrote this to stack overflow. I copied and edited it so I can consult it later.


Step 1: Create github repository

Step 2: Copy the URL of the github repository

Step 3: Add remote for the github repository (git remote add origin "copied repo URL")

Step 4: Add file (git add * or git add . or git add file name)

Step 5: git commit -m "your message"

Step 6: Push it (git push origin "branch name") (for example: git push origin master)
